# Offline License Activator  

The **PDMPublisher Offline License Activator** (activator.exe) is a secure, standalone console utility designed for environments where internet connectivity is restricted or prohibited.

![offlineactivator](/images/offlineactivator.png)

This tool enables authorized administrators to consume a PDMPublisher license without requiring:

- Internet access  
- Cloud validation  
- External HTTP communication  
- Telemetry or outbound traffic  

It is specifically intended for:

- Defense contractors  
- Aerospace programs  
- Government-secured networks  
- Air-gapped facilities  
- Isolated manufacturing systems
